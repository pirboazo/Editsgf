maxiGos

maxiGos de François Mizessyn est un ensemble de lecteurs sgf utilisant html5 et javascript permettant d’afficher des diagrammes, parties ou problèmes de go dans une page web.

Installation.

Il y a deux instalations possibles de maxiGos dans chaque cas le navigateur doit être capable d’interpréter html5 et avoir javascrit activé.

  • Version lecteur simple
  • Version Complete nécessite l’installation de php sur le serveur.

vous trouverez sur la page de téléchargement toutes les informations utiles à a mise en oeuvre.

Sur ce site j’ai utilisé

  • maxigos-minimalist-edit pour l’éditeur
  • maxigos-classic-game pour les lecteurs
